# TP2

## Niveau 1

Le joueur est un robot qui obtient la conscience et doit s’échapper du bâtiment.  Dans le niveau 1 le robot se réveille dans un petit laboratoire. Pour s’échapper du laboratoire il devra résoudre un casse-tête pour ouvrir le coffre-fort du laboratoire qui contient un pied de biche. Le pied de biche servira de clé pour ouvrir la porte pour sortir du laboratoire.

## Niveau 2

Une fois le joueur sorti du laboratoire le joueur sera confronter à une grande pièce avec des ligne d’assemblement.   


Developed with Unreal Engine 5
